# Security Policy

## Supported Versions

Currently, only the latest major release of NeuroForge AI is actively supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability within NeuroForge AI, please reach out to the project maintainers directly via an issue or secure channel. We take all security vulnerabilities seriously and will work to address them promptly. Do not disclose the vulnerability publicly until a patch has been released.
