# NeuroForge AI

NeuroForge AI is an advanced code intelligence, weakness detection, and skill insights extension powered by the NeuroForge backend. 

**Developed by Jibin Jose**

## Features
- **Inline Diagnostics:** Detects structural weaknesses and architectural flaws in your codebase and highlights them with inline squiggly lines.
- **Intelligent Suggestions:** Provides actionable recommendations to improve clean code scores and reduce complexity.
- **Typing Time Analysis:** Tracks your active coding time and correlates it with logic complexity.
- **Workspace Analysis:** Scans your entire workspace to provide global metrics and architecture-level insights.

## Commands
* `NeuroForge: Analyze Current File` - Analyzes the currently active file for weaknesses and suggestions.
* `NeuroForge: Analyze Full Workspace` - Analyzes the entire workspace to provide global code quality metrics.

## Requirements
Requires the NeuroForge AI Engine running locally (default: `http://localhost:8000`). Make sure the AI Engine FastAPI server is active.
